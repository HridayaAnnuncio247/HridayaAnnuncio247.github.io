Movie Recommendation System Project


  • Link to the rec system deployed on Render.

  • Link to Github Repository with all the codes.

  • Link to youtube playlist. These are some videos I created to show my process of working on the project. More are yet to come.

  • A few points of description

    • Created a set up page where users could select their initial preferences. This was done to help with the cold start problem.
    • Instead of using ALDI for the cold start problem, I averaged the movie embeddings to create the initial user embedding.
    • Cast and crew for movies was represented in the movie embeddings as word2Vec vectors.
    • For first recommendation, calculated reward just by using cosine similarity.
    • Created a top k reward movies for the user.
    • Used the hyperparameter pnew to explore movies besides the top k.
    • Used Epsilon Greedy algo within the top k with a 1-pnew probability.
    • Used LinUCB (contextual bandits) instead of DQN due to lack of user data.
    • LinUCB only helped with the accuracy reward. However, in the future I would like to inculcate the fairness and exploration rewards to understand if a user prefers to explore movies faraway from their likings too and in what proportions.
    • Users can like(reward 1) and dislike(reward -1) movies. (Movies with no like or dislike assigned reward of 0)
